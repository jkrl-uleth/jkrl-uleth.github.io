  1. Difference between CSS padding and margin

    From this question I learned that in the CSS box model there are four layers: the content, padding, border, and margin. The border is in between the padding and the margin. Styles like background color are not applied to the margin but are applied to padding

  2. Linking a JavaScript event listener to a class

    In this question I learned that if you use the function getElementsByClassName you can select an array-like object of DOM objects that all have the selected class name. You can then iterate through these to add event listeners to all of them. However, I didn't end up needing this because I only added one event listener.
